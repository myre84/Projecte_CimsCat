# Projecte_CimsCat
Plataforma web per explorar, planificar i registrar activitats de muntanya a Catalunya. Vue 3 + Node.js + PostgreSQL

## Backend: stats, challenges i badges

S ha implementat al backend el sistema d estadistiques d usuari, reptes (challenges) i badges persistents. Veure `docs/CIMSCAT_BACKEND_STATS_CHALLENGES_BADGES_IMPLEMENTACIO_I_VALIDACIO.md` per instruccions d instal.lacio, seed i proves manuals.

## Backend: routes planificades

S ha implementat el modul de rutes planificades amb punts ordenats i visibilitat publica. Veure `docs/CIMSCAT_BACKEND_ROUTES_IMPLEMENTACIO_I_VALIDACIO.md` per la guia completa.
